# How to run, step by step

### 1. Install project dependencies
`npm install`

### 2. Create a file with name '.env' in the root folder and add environment variables:
`PORT={****}`
`STRIPE_SECRET_KEY='****'`

### 3. Run app in background
`npm run dev` = `nodemon server.js`

# Dependencies
`npm install express`
`npm install cors`
`npm install nodemon`
`npm install stripe`
`npm install dotenv`
